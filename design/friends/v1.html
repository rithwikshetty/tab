Friends tab & non-group balances

A new Friends tab sits first in the bar: Friends · Trips · Activity · Settings. It answers "how much do I owe each person overall" by netting every trip plus non-group expenses, per currency. Tapping a person shows the breakdown by source and a shared history. Settle-up is per source — the global number is derived, never its own ledger.

Settle-up — pick source
"Settle up" lists each source with a live balance. Tapping one opens the existing settle-up form scoped to that trip (or the non-group container). The global net updates as a result.

How it hangs together

The Friends tab is a read model over data that already exists, plus the new non-group container.

  • Net per person per currency = sum of that person's pair balances across every shared trip, plus the non-group balance. No FX blending — currencies stay on separate lines.
  • Friend detail breaks the net back out by source and shows a unified timeline of shared expenses and settlements, each tagged with its origin.
  • Settle-up is per source. You settle the non-group balance or a specific trip's balance; each writes a settlement in that context. The global number is derived and updates as a consequence — deliberately avoiding Splitwise's confusing cross-group "settle all".
  • FAB = "Add expense". From Friends it defaults to non-group; from a trip it's scoped to that trip. "Create trip" moves to the Trips tab's nav bar.
